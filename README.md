# amazon-clone
A basic Amazon homepage clone built with HTML and CSS for learning purposes.

# Project Overview
This project is a **basic Amazon homepage clone**, created as a **practice project** to improve frontend development skills.  
It is built using **HTML and CSS** and demonstrates a simple layout inspired by Amazon’s homepage.

# Features
- Header with logo and search bar  
- Navigation menu  
- Product listings with images   
- Footer section  
- Clean design for desktop view  

# Technologies Used
- HTML5  
- CSS3  

# Purpose of This Project
The purpose of this project is to demonstrate:  
- Understanding of HTML page structure  
- CSS styling, layouts, and responsiveness  
- Frontend development skills inspired by real-world websites  

# Folder Structure
- **index.html** — Main HTML page  
- **style.css** — Stylesheet for layout and design  
- **images/** — Folder containing all image assets used in the website  
- **output-screenshots/** — Folder containing desktop view screenshots of the website
